# Zeke: Autopilot Operations Plan

**Philosophy:** Run the business with AI agents, humans as orchestrators and quality gatekeepers.

---

## Executive Summary

Zeke is designed to run 90% on autopilot using 29 specialized AI agents. This document outlines the daily, weekly, and monthly operations required to keep the business growing with minimal human effort.

**Time Commitment:** 5-10 hours/week after initial setup
**Primary Role:** Orchestrator + quality gatekeeper + strategic partnerships

---

## The Autopilot Stack

### Tier 1: Fully Automated (Zero Human Touch)
These run 24/7 with no intervention:

| Agent | Function | Runs |
|-------|----------|------|
| Homeowner Q&A Agent | Answers user questions | Real-time |
| Home Profile Agent | Maintains user home data | Real-time |
| Diagnostic/Triage Agent | Assesses issue severity | Real-time |
| Product Recommendation Agent | Suggests DIY products | Real-time |
| Proactive Outreach Agent | Seasonal reminders, follow-ups | Daily |
| Content Flywheel Agent | Logs questions, identifies patterns | Continuous |

**Human Role:** Monitor metrics dashboard, review flagged items only

---

### Tier 2: Human-Approved (Light Touch)
Agent does the work, human approves before action:

| Agent | Function | Human Time | Frequency |
|-------|----------|------------|-----------|
| Content Generation Agent | Drafts SEO articles | 15 min review | 2-3x/week |
| Content QA Agent | Fact-checks content | Auto-approves if score >90% | Continuous |
| Inspector Outreach Agent | Drafts personalized emails | 5 min review batch | Weekly |
| Agent Outreach Agent | Drafts agent pitches | 5 min review batch | Weekly |
| Pro Matching Agent | Matches pros to jobs | Auto after network is built | Real-time |

**Human Role:** Quick approve/reject + occasional quality spot checks

---

### Tier 3: Human-Initiated (Strategic)

| Agent | Function | Human Time | Frequency |
|-------|----------|------------|-----------|

## Daily Operations (15-30 minutes)

### Morning Routine (10 min)
**Dashboard Review:**
```
LOGIN → Check overnight metrics:
├── New users (target: 5-10/day after Week 12)
├── Questions answered (target: 50-100/day)
├── Pro referrals made (target: 1-2/day)
├── Content published (target: 2-3/week)
└── Flagged items (errors, low confidence answers)
```

**Actions:**
- Review any flagged Q&A responses (low confidence or safety concerns)
- Approve any pending content (if Content QA flagged for human review)
- Check for inspector/agent outreach responses

**Tools:** Single dashboard showing all key metrics + flags

---

### Evening Check (5-10 min)
**Human-in-Loop Queue:**
```
QUEUE REVIEW:
├── Content ready for publish: Approve/Edit/Reject
├── Outreach messages: Approve/Edit/Hold
├── Pro referral quality: Spot check 2-3
└── Partnership opportunities: Flag for follow-up
```

**Actions:**
- Batch approve content (1-2 articles)
- Batch approve outreach (5-10 messages)
- Note any issues for agent retraining

## The Content Flywheel (Autopilot)

### How It Works:
```
USER QUESTIONS (continuous)
    ↓
Content Flywheel Agent (daily):
├── Logs all questions
├── Identifies patterns
├── Scores by: volume, uniqueness, SEO potential
└── Prioritizes top topics
    ↓
Content Generation Agent (2-3x/week):
├── Generates SEO-optimized article
├── Includes affiliate links
├── Adds internal links
└── Submits to QA
    ↓
Content QA Agent (immediate):
├── Fact-checks against knowledge base
├── Checks for safety issues
├── Scores quality (0-100)
└── Auto-publishes if >90, else flags for human
    ↓
PUBLISHED → Google indexes → Ranks → New users find → Ask questions → Loop
```

**Human Role:**
- Approve high-priority topics (5 min/week)
- Review flagged content (15 min/week)
- Monitor traffic/rankings (10 min/week)

**Result:** 10-15 articles/month published with minimal effort

## Inspector Outreach (Semi-Autopilot)

### How It Works:
```
WEEKLY:
Inspector Outreach Agent:
├── Pulls list from ASHI directory (South Shore filter)
├── Researches each inspector (reviews, specialties, activity)
├── Generates personalized email
├── Scores warm vs cold
└── Queues for human approval
    ↓
HUMAN REVIEW (30 min/week):
├── Batch approve 10-15 messages
├── Adjust any messaging
├── Identify very warm leads for personal outreach
└── Send
    ↓
Agent tracks responses:
├── Opened/replied/interested
├── Schedules demos automatically (Calendly)
└── Follows up on no-response after 1 week
    ↓
DEMO SCHEDULED → Human conducts demo → Agent handles onboarding
```

**Human Role:**
- Quick approve outreach messages (5 min)
- Conduct demos (30 min each, 2-3/week)
- Personal follow-up on hot leads

**Result:** 5-10 new inspector conversations/month

## Pro Network Management (Autopilot)

### How It Works:
```
CONTINUOUS:
Pro Network Management Agent:
├── Tracks every referral
├── Monitors lead outcome (did they hire? rating?)
├── Identifies quality issues
└── Maintains pro ratings
    ↓
Quality Thresholds:
├── Rating <3.5 → Flag for review
├── 3 poor outcomes → Remove from network
├── 5-star ratings → Promote in matching
└── No availability → Deprioritize
    ↓
WEEKLY REPORT:
├── Pro performance summary
├── New pros needed (gaps in coverage)
├── Revenue by pro
└── Human reviews + takes action
```

**Human Role:**
- Review flagged pros (10 min/week)
- Recruit new pros in gap areas (30 min/week)
- Maintain high-quality relationships

**Result:** Self-maintaining network of 20-50 vetted pros

## Agent Retraining & Optimization

### Monthly Agent Reviews
**For each agent:**
```
PERFORMANCE CHECK:
├── Volume (how many tasks processed)
├── Quality (success rate, user ratings)
├── Speed (latency, throughput)
└── Cost (API usage)
```

**If underperforming:**
- Review recent outputs
- Identify failure patterns
- Update prompts or knowledge base
- A/B test improvements

**Time:** 1 hour/month per agent (rotate focus)

## Crisis Management

### Scenarios That Need Human Intervention:
1. **Safety Issue:** Agent gives dangerous advice
   - Alert threshold: Any safety-related keyword flagged
   - Response: Immediate human review + fix

2. **PR Crisis:** Negative press or viral complaint
   - Alert: Social media monitoring
   - Response: Manual response + agent pause if needed

3. **Pro Network Failure:** No pros available for urgent job
   - Alert: Pro availability <2 in any category
   - Response: Manual pro recruitment

4. **Revenue Drop:** 50% decrease week-over-week
   - Alert: Automated weekly comparison
   - Response: Root cause analysis
